What is Node.js?

February 25th by Evan

This blog post will show you the basics of Node.js.

Defining Node.js

Node.js is an open source, cross-platform runtime environment for server-side and networking applications.

It uses JavaScript and can be run on OS X, Windows, Linux, and more.

The Growth of Node.js

The reason that many people are seeing why Node.js is more popular than other alternatives is because it is an asynchronous, event-driven framework that is used to build scalable network applications. It adds JavaScript with C/C++ in order to achieve certain things other frameworks can't.

Features of Node

  • Node lets you perform other tasks while waiting to be notified when the response is available.
  • Node applications use chunking, instead of buffering the data into memory.

    • An example using Node.js to register callbacks is below.

    var http = require('http');

http.request({ hostname: 'example.com' }, function(res) {
  res.setEncoding('utf8');
  res.on('data', function(chunk) {
    console.log(chunk);
  });
}).end();

Node.js allows the use of a non-blocking I/O API as mentioned above, which lets users work on multiple things at a time. Besides tasks that depend on the I/O to complete it's process, every other task can be run simaltaneously.
